neXtProt most recent human PEFF sequence database has been updated to the most recent HUPO-PSI PEFF format
(which is subject to change yet again).  If you want to perform a PEFF search, you can search against this
database:  ftp://ftp.nextprot.org/pub/current_release/peff/nextprot_all.peff.gz

Use a tool like 7-zip on Windows or gunzip on linux to uncompress this .gz archive.
